# Nest_API_Adriana
API Nest de Pokemon y peliculas

---

## Pruebas rápidas
1) POST http://localhost:3001/movies/seed
2) GET  http://localhost:3001/movies/search/title?q=star
3) GET  http://localhost:3001/movies/search/year-between?min=1990&max=2010 

4) POST http://localhost:3001/pokemon/seed
5) GET  http://localhost:3001/pokemon/search/type?type=water
6) GET  http://localhost:3001/pokemon/search/min-hp?min=100
